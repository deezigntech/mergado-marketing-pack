== Description ==

[Mergado Pack for WooCommerce](https://pack.mergado.com/woocommerce) is a **free tool** that allows you to advertise on comparison shopping services. It helps you **create better XML data files** by exporting product feed from your WooCommerce store in the [Mergado XML specification](https://www.mergado.com/product_xml_specification.pdf). Such data is useful for linking to Mergado app, where it can be converted to **hundreds of other formats** (for comparison shopping engines like Shopping.com, for AdWords or Facebook or for other online shopping platforms).

Mergado Pack plugin is packed with many **marketing functions** that will help you improve your advertising. Support for Facebook Pixel, Glami piXel and new Glami TOP reviews and much more.

#SUPPORT FOR GENERATING PRODUCT CODES FROM OTHER PLUGINS

&#9989; [EAN for WooCommerce](https://wordpress.org/plugins/ean-for-woocommerce/)
&#9989; [WPSSO Product Metadata (GTIN, UPC, EAN, ISBN, MPN, Global Identifier) for WooCommerce](https://wordpress.org/plugins/wpsso-wc-metadata/)

== Installation ==

Installation is standard as for other Woocommerce plugins. See our videotutorial:

https://youtu.be/L4w3ZfumBUU

and follow these steps:
1. In your store administration go to Menu and click **Plugins → Add new**.
2. Search for 'mergado' keyword. When the Mergado marketing pack plugin appears, click **Instal now**.
3. After the installation is complete, click **Activate**.
4. You will find the module both in the menu on the left as **Mergado** and under **Installed plugins**.

== Frequently Asked Questions ==

= What can I use the Marketing Pack for Woocommerce for? =

The module allows you to **export data from your online store in XML format**, in [Mergado XML specification](https://www.mergado.com/sites/default/files/users/documents/mergado-product-xml-1-4_en.pdf). Such data can be used to connect your store to [Mergado](https://www.mergado.com), where you are able to convert your product data to [over 200 different formats](https://www.mergado.com/supported-channels) for **comparison shopping services** such as Google Shopping, Facebook, Glami, etc., or for **advertising systems** like Google Ads, Facebook, or for different **eCommerce solutions** like e.g. Shoptet, etc.

= How much does the module cost? =

The module is **for free**.

= How do I install the module? =

The module is installed in a standard way, same as other WordPress plugins. Watch the video tutorial above with a step-by-step installation guide.

= How do I uninstall the module? =

In a standard way, same as other Woocommerce plugins are uninstalled.

= How does the module updates work? =

The module **automatically checks for the latest version** and if there is one, it will display a notification in the administration of your store that an update is available.

= Do I need to set up anything after the module installation? =

Yes, the module needs to be set up in order to function properly. Follow these 2 steps:
1. On the "Cron" tab you will find a link to XML feed generation. For a one-time generation click this link and the feed will be generated (or re-generated, if it already exists). To schedule the generation in regular intervals copy the link and set it up in your web hosting to tasks launched by cron (more on cron setup in the next Q&A)
2. On the "XML feed" tab you can find the XML feed URL address and use the link to create a new export in Mergado.

= I need help with the module setup. Who can I turn to for help? =

If you need any help with installation and the first setup of the module, contact our support at [mergado@mergado.com](mailto:mergado@mergado.com)

We would appreciate if you let us know at [mergado@mergado.com](mailto:mergado@mergado.com)
